# Parking Locator
- Clone this repo
```bash
gh repo clone hilmizul/parking-locator
```

- Go to folder
```bash
cd parking-locator
```

- Install dependencies
```bash
npm install
```

- Run
```bash
npm run start
```

- Go to ```http://localhost:4000```

### Error?
- Create database: ```parking_locator```
- Create table: ```parking```
| Field | Type |
|---|---|
| #id | int(11) |
